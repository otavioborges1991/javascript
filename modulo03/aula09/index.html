DOM

O DOM (Document Object Model) é uma interface de programação para documentos HTML e XML. Ele representa a estrutura do documento como uma árvore de nós, onde cada nó é um objeto que pode ser manipulado por linguagens de programação, como JavaScript. Com o DOM, é possível acessar e modificar o conteúdo, a estrutura e o estilo dos documentos de forma dinâmica.
