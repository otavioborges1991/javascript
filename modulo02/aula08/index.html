Operadores Relacionais, logicos e ternario.

Operadores de Comparação, ou Relacionais.

  • == igualdade.
  • != diferente.
  • === igualdade estrita.
  • !== diferente estrito.
  • > maior que.
  • < menor que.
  • >= maior ou igual.
  • <= menor ou igual.

Os operadores de comparação de identidades retornam true somente se o tipo e valor forem iguais

Os operadores de comparação de igualdade retornam true se os valores forem iguais, mas não necessariamente o tipo.

Operadores Lógicos.

  • && E lógico
  • || OU lógico
  • ! NÃO lógico

Operadores lógicos são usados para combinar expressões booleanas. Eles retornam true ou false.

O operador && retorna true se ambas as expressões forem verdadeiras.

O operador || retorna true se pelo menos uma das expressões for verdadeira.

O operador ! inverte o valor booleano da expressão.

a ordem de precedência dos operadores lógicos é a seguinte:

  1. ! (NÃO lógico)
  2. && (E lógico)
  3. || (OU lógico)

Operador Ternário.

O operador ternário é uma forma concisa de escrever uma expressão condicional. Ele tem a seguinte sintaxe:
condição ? valor_se_verdadeiro : valor_se_falso

O operador ternário avalia a condição. Se for verdadeira, retorna o valor_se_verdadeiro; caso contrário, retorna o valor_se_falso.

É uma maneira compacta de escrever uma estrutura condicional simples.
